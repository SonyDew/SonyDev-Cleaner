# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, adapted for a small application repository.

## [1.0.0] - 2026-03-16

### Added

- Initial public open-source release
- WPF desktop application with glass-style UI
- Safe Cleanup workflow with grouped targets and report export
- Home dashboard with stats, schedule toggle, and quick actions
- Large Files analysis page
- Activity page with styled logs
- Startup Manager page
- Disk Usage page
- Tray icon, background scan mode, portable mode, and scheduled weekly scan
- GitHub-ready documentation, contribution guidelines, security policy, issue templates, and workflow
