Initial public open-source release of **SonyDev Cleaner**.

## Release Assets

- `SonyDevCleaner-Setup-1.0.0-win-x64.exe` - Windows installer built with Inno Setup
- `SonyDevCleaner-Portable-1.0.0-win-x64.zip` - portable package that stores data next to the executable when `portable.txt` is present

## Highlights

- WPF desktop application for Windows 10 and Windows 11
- Safe Cleanup dashboard with grouped scan results and exportable reports
- Large Files analyzer, Activity page, Startup Manager, and Disk Usage page
- Tray support, background scan mode, portable mode, and weekly scheduled scan
- GitHub-ready repository setup with GPL-3.0 licensing and contribution docs

## Included Modules

- Home dashboard
- Safe Cleanup
- Large Files
- Activity
- Startup Apps
- Disk Usage

## Repository Setup

- README with screenshots
- changelog and roadmap
- publishing guide
- issue templates and PR template

## Platform Notes

- Windows-only
- Targets `net8.0-windows`
- Cleanup design is preview-first and intentionally avoids registry cleaning or hidden tweaks
