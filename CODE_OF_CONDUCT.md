# Code of Conduct

## Our Standard

We want this repository to be welcoming, useful, and respectful.

Examples of expected behavior:

- be respectful in issues, reviews, and discussions
- assume good intent and discuss technical disagreements directly
- give actionable feedback
- focus on the software, not the person

Examples of unacceptable behavior:

- harassment or personal attacks
- discrimination of any kind
- trolling, baiting, or repeated bad-faith argument
- publishing private information without permission

## Enforcement

Project maintainers may remove comments, close discussions, reject contributions, or block participation when behavior violates this code of conduct.

## Scope

This applies to issues, pull requests, discussions, code review, and other project communication channels.
