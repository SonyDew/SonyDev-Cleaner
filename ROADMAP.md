# Roadmap

This roadmap is intentionally practical. SonyDev Cleaner should grow as a safe, transparent Windows utility, not as an aggressive "optimizer".

## Current Focus

- Keep the WPF application stable and understandable
- Improve scan quality without making cleanup behavior riskier
- Polish documentation and repository quality for contributors

## Next

- Add better per-target exclusion support
- Improve report export formats
- Add clearer status messaging for restricted and elevation-required targets
- Refine disk usage analysis performance on large drives
- Add more tests around non-destructive service logic

## Later

- Multi-profile browser cache detection
- Better analytics for developer cache targets
- More granular scheduled task options
- Optional packaging and signed release distribution workflow

## Explicit Non-Goals

- Registry cleaning
- Hidden system tweaks
- "Boost" features that change Windows behavior without clear user intent
- Bundled software or adware-style integrations
